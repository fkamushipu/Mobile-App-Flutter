# Mobile-App-Flutter
This is a mobile application template using flutter

Using Visual Studio Code as an editor.

1. Go to view - click on Command Pallette 
2. Search for Flutter: New Project.
3. Select Application.
4. Create or select the parent directory for the new project folder.
5. Enter a project name, such as flutter_application, and press Enter( all lowercase and can only be separated by an underscore).
6. Wait for project creation to complete and the main.dart file to appear.
